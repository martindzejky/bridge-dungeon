# Bridge Dungeon

Endless bridge runner in a dark dungeon. **Made in 1 day as a challenge**.
Control using arrow keys or A/D keys to move.

## Tools

- [Godot](https://godotengine.org)
- [Aseprite](https://www.aseprite.org)
- [Sfxr](https://sfxr.me)
- [Suno](https://suno.com)
